Lindisfarne 793

Visual narrative

Interactive choice-based visual narrative used to engage children to learn about history.

The image above showcases the scenes created for the visual narrative, developed using the Unreal Engine 5. In this screenshot, the characters, or actors, are depicted in their default 'T-pose,' as it was taken from the editor overview rather than a perspective that a user would experience 'in-game.'

The project incorporated an interaction tool known as the 'Leap Motion,' providing a VR-like hand control experience without the need for goggles. This controller was instrumental in engaging users in both interactive aspects and immersing them in the historical narrative, allowing them to feel like an integral part of the story.

This project marked my first experience with the Unreal Engine, and my primary role was coding the choice-based narrative elements. Although I also contributed to some animations, my main focus was on coding. Initially, working with the Unreal Engine appeared to be a formidable task, but as I familiarized myself with the interface, I found it to be not too dissimilar from working with Unity. While there were discernible differences, the project proved to be an enjoyable learning experience.
